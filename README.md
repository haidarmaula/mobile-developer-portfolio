### 🐦 WatsdaBird — Bird Sound Classification App

A personal end-to-end project: a cross-platform mobile app that identifies bird species from audio recordings using a CNN deep learning model. Built the entire stack — from Flutter frontend to Python ML backend — solo.

**Tech Stack**
- Flutter + Dart
- flutter_sound (audio recording & playback)
- HTTP (backend API integration)
- Python + Flask (backend)
- TensorFlow + Keras + librosa (ML inference)
- Docker + Gunicorn (backend deployment)

**Highlights**
- Designed and built the full mobile-to-backend pipeline independently
- Users can record live audio or upload audio files for identification
- **97% classification accuracy** across 4 bird species
- Model trained on 12,510 audio samples with audio preprocessing and spectrogram generation
- CNN inference completing in **~2.3s** per prediction
